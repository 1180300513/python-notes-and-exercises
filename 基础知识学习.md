## 3.条件控制语句

### 3.1 if语句

```python
# if 条件:
#     代码块

x = 10
if x > 5:
    print("x 大于 5")

# if 条件:
#     代码块1
# else:
#     代码块2

x = 3
if x > 5:
    print("x 大于 5")
else:
    print("x 小于或等于 5")

# if 条件1:
#     代码块1
# elif 条件2:
#     代码块2
# else:
#     代码块3
x = 8
if x > 10:
    print("x 大于 10")
elif x > 5:
    print("x 大于 5 且小于等于 10")
else:
    print("x 小于或等于 5")
```
